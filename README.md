# RC BOT
FURY ROAD ROBOTIC COMPETITION 2023

COMPONENTS:
1. Arduino Uno
2. L298N Motor driver
3. 100rpm geared 12v motor
4. tattu 11.2 lipo 3 cell battery
5. fly sky CT6B transmitter and receiver
